# פלופ! 🚽 צלילה אל הביוב

משחק arcade אינסופי לנייד — אתה גולש במורד צינור הביוב כקקי גיבור 💩, אוסף תירס 🌽 וחומק ממכשולים.
מושלם לרגעים הגדולים של היום. 😌

## איך משחקים
- **גוררים את האצבע** ימינה ושמאלה כדי לתמרן (שליטה ביד אחת).
- אוספים **🌽 תירס** לנקודות ו-**⭐ כוכבים** לבונוס.
- חומקים מ-🪠 פומפות, 🧽 ספוגים, 🪥 מברשות, 🦆 ברווזים, 🧻 נייר.
- איסוף רצוף בונה **קומבו** (עד x9). ככל שיורדים עמוק יותר — מהיר וקשה יותר.
- השיא נשמר במכשיר.

## טכנולוגיה
קובץ HTML5 בודד, ללא תלויות — Canvas, Web Audio, ושמירה מקומית.
דמות הקקי מצוירת בוקטור, רקע חלק עם תנועת בוקה (ללא הבהובים).

## הרצה מקומית
פשוט פותחים את `index.html` בדפדפן, או:

```bash
npx serve .
```

נבנה עם [Claude Code](https://claude.com/claude-code).
